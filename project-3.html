Ingeniería

En ingeniería tenemos 3 áreas a cubrir: el diseño, el análisis y la manufactura de un prototipo de Formula 1 a escala. En cada área se utiliza un software especializado.

Diseño

Diseño

Para el diseño se utiliza un software CAD (Diseño asistido por computadora), en nuestro caso, utilizamos Fusion 360-Autodesk en el que se diseña el prototipo a través de un modelado 3D.

Análisis

Para el diseño se utiliza un software CAD (Diseño asistido por computadora), en nuestro caso, utilizamos Fusion 360-Autodesk en el que se diseña el prototipo a través de un modelado 3D.

Manofactura

Manofactura

Para la manufactura primero se procesa la geometría antes analizada en un software CAM (Manufactura Asistida por Computadora). El software CAM que utilizamos es "Fusion 360" y Siemens NX en donde se simula un maquinado virtual en el que se puede establecer la herramientas a utilizar y la velocidad que una máquina llevará, además es donde se genera el "Còdigo G".
